C语言

发表于 2019-05-08

C语言程序的结构：

1.一个程序由一个或多个源程序文件组成：（1）预处理命令：如#include<stdio.h>；（2）全局声明；（3）函数定义；

2.函数是C程序的主要组成部分：一个C语言程序中必须包含一个main函数（且只能由一个）；

3.一个函数包括两个部分：（1）函数首部：包括函数名，函数类型，函数属性，函数参数（形式参数）名，参数类型；如int属于函数类型，max属于函数名，（int属于函数参数类型，x，属于函数参数名，int属于函数参数类型，y）属于函数参数名。也就是int max(int x,int y)；（2）函数体：包括声明部分，执行部分。

4.程序总是从main函数开始执行的。

5.程序要求计算机完成的操作是由函数中的C语句完成的。

6.每个数据声明和语句的最后必须有一个分号。

7.C语言本身不提供输入输出语句。

8.程序应当包含注释。

程序设计的任务：

1.问题分析

2.设计算法

3.编写程序

4.对源程序进行编辑，编译和连接，得到可执行程序

5.运行程序，分析结果

6.编写程序文档

算法—程序的灵魂

程序=算法+数据结构

数据结构：在程序中要指定用到哪些数据，以及这些数据的类型和数据的组织形式。

算法：要去计算机进行操作的步骤。

计算机算法可分为两大类：数值运算算法和非数值运算方法。

算法的特性：1.有穷性，2.确定性，3.有零个或多个输入，4.有一个或多个输出，5.有效性。

可以用自然语言表示算法，用流程图表示算法，用N-S流程图表示算法。

转义字符及其作用：

\’表示输出单撇号字符’；

\a表示警告，会产生声音或视觉效果；

\b表示退格，将光标当前位置后退一个字符；

\f表示换页，将光标当前位置移到下一页开头；

\n表示换行，光标当前位置移到下一行开头；
